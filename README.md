# Özür Sitesi ❤️

İnteraktif web bileşenleri ve modern CSS yapıları kullanarak geliştirdiğim, kullanıcı deneyimini (UX) esprili ve eğlenceli hale getirmeyi amaçlayan bir frontend projesi.

🔗 **[Canlı Demo İçin Tıklayın](https://lutfen-beni-affet.netlify.app)**

## 🚀 Proje Hakkında
Bu proje, React'in state yönetimi ve temel hook yapısını kullanarak kullanıcı ile "oyunbaz" bir etkileşim kurmak amacıyla tasarlanmıştır.

### Öne Çıkan Özellikler
- **İnteraktif Kaçış Mekanizması:** "Hayır" butonu, mouse ile üzerine gelindiğinde veya tıklandığında ekranda rastgele konumlara kaçar.
- **Duygusal Animasyonlar:** Affedilme durumunda tetiklenen konfeti efektleri ve dinamik ayıcık GIF'leri.
- **UX Optimizasyonu:** Tarayıcı "Geri" tuşu koruması (popstate yönetimi) ile kullanıcının uygulama içinde kalması sağlanmıştır.
- **Responsive Tasarım:** Tailwind CSS ile mobil ve masaüstü cihazlarda tam uyumlu arayüz.

## 🛠️ Kullanılan Teknolojiler
- **Framework:** [React](https://react.dev/) (Vite)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation/Effect:** `react-confetti`
- **Deployment:** Netlify
